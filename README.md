# Практика 13–14: PWA, Service Worker, Manifest, offline-first

> Для проверки PWA и Service Worker убедитесь, что приложение запускается в безопасном контексте. В браузере можно использовать `localhost`, который считается безопасным даже при обычном HTTP, либо развернуть приложение через HTTPS-сервер.
