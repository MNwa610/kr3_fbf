ПРАКТИКИ 13–14

О приложении PlanDo

PlanDo — учебный PWA-проект для заметок и задач. Приложение создано для демонстрации App Shell, кэширования статических ресурсов, динамических страниц и офлайн-поддержки.

Вернуться к задачам

Что делает это приложение

  • Сохраняет статические ресурсы в кэше при первом посещении.
  • Загружает страницы по стратегии Network First.
  • Показывает офлайн-страницу, когда сеть недоступна.
  • Хранит список задач в localStorage и работает без сети.
  • Поддерживает установку как PWA через Web App Manifest.

Почему это App Shell

App Shell — это оболочка интерфейса, которая кэшируется заранее. Она гарантирует, что приложение быстро запускается и доступно офлайн, а динамические страницы обновляются через сеть, когда это возможно.

Страница доступна офлайн

Благодаря правильной стратегии кеширования, страница «О нас» доступна без сети, если она была ранее загружена.
